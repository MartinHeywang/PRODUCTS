Le jeu

Enrichissez-vous un maximum en construisant des lignes d'assemblages dans votre usine !

Achetez, transformez des ressources afin de créer des objets de plus en plus cher : d'un simple circuit à l'ordinateur au complet !

Télécharger

Disponible dès maintenant sur Windows, Mac et Linux :

Deviens un vrai patron et installes-le jeu !

Windows (.exe) Autre (.jar)

Si vous utilisez Mac ou Linux, ou tout système d'exploitation hors Windows, choissisez la version Autre. Le fichier téléchargé s'exécute comme une application : double-cliquez simplement dessus !

Il semblerait que vous êtes sur un téléphone ! Ce jeu n'est disponible que sur PC.

Si vous rencontrez des problèmes au lancement du jeu, c'est probablement que Java n'est pas installé sur votre machine. N'hésitez pas à l'installer en allant sur java.com.

La taille du jeu risque d'augmenter avec le temps.

Open Source et gratuit :

Ce jeu est un projet, il n'est pas terminé, et n'a pas pour but de me rendre milliardaire ! Il vous est donc offert : à vous de l'essayer ! Le code source est disponible sur GitHub. Cliquez sur l'image pour y accéder : .

Cliquer sur l'image pour rejoindre le serveur ! Vous y trouverez plein d'informations, une aide rapidé pour résoudre vos problèmes, sur le jeu, sur la création des plugins, sur tout ce qui va avec le jeu !

Extensions

Products est un jeu extensible : cela veut dire que n'importe qui peut rajouter du contenu, resources at appareils ! C'est malheureusement une fonctionnalité encore en développement.

Si vous souhaitez créer un plugin, veuillez vous référer ici.

Si vous voulez utiliser un plugin, ajoutez-le dans le dossier plugins(créez-le si nécessaire) à côté du fichier du jeu.

Vous êtes responsables des plugins que vous ajouter. Prenez conscience qu'ils peuvent exécuter n'importe quoi, n'importe quand, et peuvent donc nuire à votre machine. Vous devez donc faire confiance au créateur du plugin.
